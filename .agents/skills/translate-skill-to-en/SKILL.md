# translate Skill To English

This skill is designed to translate skill documents from any language to English, ensuring accurate semantic translation while preserving the original structure and format. It focuses on technical accuracy and proper English grammar patterns suitable for technical documentation. If the source language is not specified, it will attempt to detect the language and ask for confirmation.

## Skill Execution Flow

### 1. Analyze Source Skill
- Read and analyze the skill document in the source language
- Determine the source language (detect if not specified)
- Understand the skill's purpose, structure, and content
- Identify technical terms and concepts that require specialized translation
- Preserve the original document structure and formatting

### 2. Language Confirmation (if not specified)
- If the source language is not specified, attempt to detect it
- Ask for confirmation of the detected language
- Allow the user to specify a different language if needed

### 3. Translation Process
- Translate content while maintaining semantic meaning
- Apply correct English grammar and technical writing patterns
- Avoid literal word-for-word translation
- Ensure technical terms are accurately translated
- Maintain the original document structure and format

### 4. Quality Assurance
- Review translated content for accuracy
- Verify technical terminology is correctly translated
- Ensure proper English grammar and syntax
- Confirm document structure is preserved
- Validate that the translated skill maintains the same functionality

## Translation Principles

1. **Semantic Accuracy**: Maintain the original meaning and intent of the skill
2. **Structural Integrity**: Preserve the original document structure and format
3. **Grammatical Correctness**: Use proper English grammar and sentence structure
4. **Technical Precision**: Accurately translate technical terms and concepts
5. **Natural Language**: Ensure the translation reads naturally in English
6. **Consistency**: Maintain consistent terminology throughout the translation

## Detailed Considerations
- For technical terms, use industry-standard English equivalents
- For idiomatic expressions, use appropriate English expressions that convey the same meaning
- Maintain the original document structure, including headings, lists, and code examples
- Preserve formatting elements such as bold text, italic text, and code blocks
- Ensure the translated skill functions identically to the original
